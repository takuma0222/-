# 従業員データディレクトリ

このディレクトリには、従業員NOと氏名のマッピング情報を含むCSVファイルが格納されます。

## ファイル: employees.csv

### フォーマット
- エンコーディング: UTF-8
- 区切り文字: カンマ (,)
- 改行コード: LF
- 1行目: ヘッダー行（固定）
- 2行目以降: データ行

### 列の説明
1. EmployeeNo: 従業員NO（6桁の数字、一意）
2. Name: 氏名

### サンプル
```csv
EmployeeNo,Name
123456,山田太郎
234567,佐藤花子
345678,鈴木一郎
```

### 注意事項
- 従業員NOは6桁の数字である必要があります
- 従業員NOは重複しないこと
- 空行は無視されます
- ファイルはExcelで編集可能ですが、保存時はUTF-8を選択してください

### Excelでの編集方法
1. Excelで employees.csv を開く
2. データを編集
3. 「名前を付けて保存」→「CSV UTF-8（コンマ区切り）(*.csv)」を選択
4. 保存
